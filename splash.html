This web app will function as an educational Java linter that checks the style of users' input Java code and inform users of what the style errors are with suggestions on how to fix them. This app will be built around the style guidelines for UW CSE 14X (Intro CS1 & 2 at UW) with those students as the primary audience, but should be a helpful resource for any novice Java learners. The project proposal can be viewed here.
